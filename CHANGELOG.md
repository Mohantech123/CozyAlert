# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-07-14
### Added
- Extended NPM keywords for better SEO and discoverability.
- Added comprehensive Schema.org JSON-LD to documentation/playground for web crawlers.
- Added PNPM and Bun installation instructions to the README.

### Changed
- Downgraded TypeScript to v5.5.4 to resolve `rollup-plugin-dts` incompatibility during the build process.
- Updated all dependencies to their latest stable minor versions.

## [0.1.0] - 2026-07-14
### Added
- Initial release of CozyPopup.
- Support for Alerts, Confirms, Toasts, Offcanvas Drawers, and Queues.
- Direct DOM manipulation (Framework Agnostic).
- Built-in form generation and Regex validation.
- Complete type definitions (`.d.ts`).
